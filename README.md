# brackeys-godot-tutorial
My first Godot project following and expanding upon [Brackey's YouTube tutorial video](https://www.youtube.com/watch?v=LOhfqjmasi0&pp=ygUIYnJhY2tleXPSBwkJkQoBhyohjO8=).
# Extra features:
- Finite State Machine for controlling the player.
	- Double Jump State.
- AudioManager for handling sound effect logic.
